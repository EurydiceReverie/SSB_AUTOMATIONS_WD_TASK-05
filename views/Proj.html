Welcome! to the Eurydice project page. This project is focused on machine learning and aims to identify sound using advanced algorithms. To explore the project further, please visit the GitHub repository.

|- About Eurydice!

Eurydice is a versatile music recognition tool written in Python that can identify and tag unknown audio files by using multiple services.
